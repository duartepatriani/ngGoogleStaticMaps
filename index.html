ngGoogleStaticMaps

Directive For Google Static Maps.

Getting Started

  1. bower: bower install --save ngGoogleStaticMaps
  2. Include bower_componets/ngGoogleStatisMaps/ngGoogleStaticMaps.min.js.
  3. Include ngGoogleStaticMaps in your module dependencies.
  4. Use the static-map directive.

Code

<static-map
        map-width="400"
        map-height="400"
        zoom="14"
        address="eiffel tower"">
</static-map>
